# Drawer / Sheet 抽屉

> 组件规范主源。与 `../BASE.md` 配合使用。
> 系统ID：`component.drawer-sheet`。本文件人工维护；`../COMPONENTS.md` 由同步脚本自动聚合生成。

<!-- DS_SPEC_START -->

基于 Shadcn/ui `Sheet`。从右侧弹出，遮罩 `rgba(0,0,0,0.3)`。

| 项目 | 值 |
|---|---|
| 宽度 | 378–480px |
| 标题栏高度 | 56px |
| 内容区内边距 | 24px |
| 背景 | `#FFFFFF` |
| 左侧阴影 | `shadow.lg` |

标题左对齐，× 关闭右对齐；底部操作栏固定（可选）；内容区独立滚动；Esc 或点击遮罩关闭。
