# Text Link 文字链接

> 组件规范主源。与 `../BASE.md` 配合使用。
> 系统ID：`component.text-link`。本文件人工维护；`../COMPONENTS.md` 由同步脚本自动聚合生成。

<!-- DS_SPEC_START -->

内联使用，无边框无背景。字号 12–14px。

| 状态 | 颜色 | 装饰 |
|---|---|---|
| 默认 | `#0067D1` | 无下划线 |
| hover | `#2E86DE` | 下划线 |
| disabled | `#BFBFBF` | 无 |

规则：用于文档跳转、帮助入口；不用于主操作按钮。
