# Template：插件配置面板

适用场景：Figma 插件、浏览器插件；单任务配置、检查、导出、同步面板；宽度 320–480px 的窄面板 Demo。

## 首屏结构（必须包含）

1. **面板头部**：插件名称、当前文件 / 项目上下文、关闭 / 更多图标
2. **状态摘要**：连接状态、检查对象、最近同步时间或任务进度
3. **配置分组**：表单项、选择器、Switch、单选、复选；按功能分组，不塞成一个长表单
4. **预览 / 结果区**：将影响的页面、组件数量或检查结果摘要
5. **底部操作栏**：左侧次级操作，右侧唯一主按钮；有上边框

## 布局尺寸

| 项目 | 值 |
|---|---|
| 面板宽度 | 360px 或 400px |
| 面板内边距 | 16px |
| 分组间距 | 16px |
| 表单项高度 | 32px |
| 底部操作栏高度 | 48–56px（固定在底部） |

## 必备组件

`Button` · `Input` · `Select` · `Switch` · `Checkbox` · `RadioGroup` · `Separator` · `Tooltip` · `Toast`

## 禁止项

- 不生成大表格
- 不使用页面级顶部导航
- 不使用大面积插画或欢迎页作为首屏
- 底部不超过一个主按钮
