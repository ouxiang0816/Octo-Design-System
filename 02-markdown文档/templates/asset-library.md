# Template：资产库

适用场景：组件库、设计资产库、图标 / 模板 / 变量管理、可复用资源检索与导入。

## 首屏结构（必须包含）

1. **顶部导航**：全局空间和账户入口，高度 56px
2. **左侧分类**：资产类型、团队、标签、文件夹，宽度 255px
3. **工具栏**：搜索 + 类型筛选 + 状态筛选 + 排序 + 视图切换
4. **资产内容区**：表格或紧凑网格；每项必须展示名称、类型、状态、更新时间
5. **右侧详情面板**：资产预览、元信息、版本、引用关系、操作（用 `Sheet` 或内嵌面板）

## 布局尺寸

| 项目 | 值 |
|---|---|
| 左侧导航宽度 | 255px |
| 右侧详情面板宽度 | 320–360px |
| 网格卡片圆角 | ≤ 8px |
| 卡片宽高 | 固定，不因名称长短错位 |
| 工具栏高度 | 40–48px |

## 必备组件

`Input` · `Tabs` · `DropdownMenu` · `Table` · `Badge` · `Sheet` 或详情面板 · `Tooltip`

## 禁止项

- 不使用瀑布流
- 不使用大图封面主导页面
- 资产卡片必须有名称、类型、状态或更新时间，不能只展示视觉图
