# Template：项目空间

适用场景：项目列表、团队空间、文件夹管理、资产管理、项目详情入口页。

## 首屏结构（必须包含）

1. **顶部导航**：全局空间切换，高度 56px
2. **左侧导航**：空间、文件夹、项目分类，宽度 255px
3. **主内容头部**：面包屑 + 页面标题 + 主操作按钮
4. **工具栏**：搜索 + 筛选 + 排序 + 视图切换（表格 / 网格）
5. **内容区**：项目列表、表格或网格；必须包含空状态、加载态

## 布局尺寸

| 项目 | 值 |
|---|---|
| 顶部导航高度 | 56px |
| 左侧导航宽度 | 255px，菜单项高度 36px |
| 主内容区内边距 | 24px |
| 工具栏高度 | 40–48px |
| 列表行高 | 40–48px |
| 选中导航项背景 | `#EFF6FF` |

## 必备组件

`Top Navigation` · `Side Navigation` · `Breadcrumb` · `Button` · `Input` · `DropdownMenu` · `Table` 或项目卡片 · `Badge`

## 禁止项

- 不把项目空间做成品牌 Landing Page
- 不使用大尺寸项目卡片铺满首屏
- 主内容区必须有搜索或筛选能力
- 不生成营销式 Hero
